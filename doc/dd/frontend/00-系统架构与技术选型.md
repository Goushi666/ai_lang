# Web 端设计 · 系统架构与技术选型

## 1. 整体架构

```
┌─────────────────────────────────────────────────────────┐
│                    前端应用层 (Vue.js)                    │
├─────────────────────────────────────────────────────────┤
│  环境监测  │  视频巡检  │  告警中心  │  车辆控制  │  系统设置  │
├─────────────────────────────────────────────────────────┤
│              通信层 (WebSocket + Axios)                  │
├─────────────────────────────────────────────────────────┤
│                  后端服务 (FastAPI)                       │
└─────────────────────────────────────────────────────────┘
```

## 2. 技术选型

- **前端框架**：Vue 3 + Composition API
- **状态管理**：Pinia
- **路由管理**：Vue Router 4
- **UI 组件库**：Element Plus
- **图表库**：ECharts
- **HTTP 客户端**：Axios
- **WebSocket 客户端**：原生 WebSocket API
- **视频播放**：Video.js 或 HLS.js（按流协议选型）
- **构建工具**：Vite
- **代码规范**：ESLint + Prettier
