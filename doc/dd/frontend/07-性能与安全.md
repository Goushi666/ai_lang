# Web 端设计 · 性能与安全

## 1. 性能优化

- **路由懒加载**：按需加载页面组件
- **组件缓存**：使用 `keep-alive` 缓存页面状态
- **图表优化**：大数据量时使用数据抽样
- **防抖节流**：控制指令发送频率；图表更新可按业务节流
- **虚拟滚动**：告警列表数据量大时使用虚拟滚动

## 2. 网络优化

- **请求合并**：批量获取数据（在接口支持的前提下）
- **数据缓存**：使用 `localStorage` 缓存配置
- **WebSocket 复用**：单一连接多路复用
- **断线重连**：指数退避重连策略（可在封装中演进）

## 3. 安全设计

### 3.1 认证授权

- JWT Token 认证
- Token 自动刷新机制
- 路由守卫权限控制

### 3.2 数据安全

- HTTPS/WSS 加密传输
- 敏感数据加密存储
- XSS 防护（内容转义）
- CSRF 防护（Token 验证）
