# 后端设计 · 数据模型

以下为 **目标持久化** 的表结构示例（与总体设计文档一致）；MVP 阶段可用内存结构承载相同字段语义。

## 1. 传感器数据 (`sensor_data`)

```python
from sqlalchemy import Column, Integer, Float, String, DateTime
from sqlalchemy.ext.declarative import declarative_base
from datetime import datetime

Base = declarative_base()

class SensorData(Base):
    __tablename__ = 'sensor_data'

    id = Column(Integer, primary_key=True, index=True)
    device_id = Column(String(50), index=True)
    temperature = Column(Float)
    humidity = Column(Float)
    light = Column(Float)
    timestamp = Column(DateTime, default=datetime.utcnow, index=True)
```

**实现说明**：生产代码在 `SensorData` 上使用复合索引 **`(device_id, timestamp)`**（`ix_sensor_data_device_timestamp`），与单列 `index=True` 并存，便于按设备+时间窗查历史。

## 2. 告警 (`alarms`)

```python
from sqlalchemy import Boolean

class Alarm(Base):
    __tablename__ = 'alarms'

    id = Column(Integer, primary_key=True, index=True)
    type = Column(String(50))
    level = Column(String(20))  # low, medium, high, urgent
    message = Column(String(255))
    value = Column(Float)
    threshold = Column(Float)
    read = Column(Boolean, default=False)
    timestamp = Column(DateTime, default=datetime.utcnow, index=True)
```

## 3. 车辆状态 (`vehicle_status`)

```python
from sqlalchemy import Boolean

class VehicleStatus(Base):
    __tablename__ = 'vehicle_status'

    id = Column(Integer, primary_key=True, index=True)
    device_id = Column(String(50), index=True)
    mode = Column(String(20))  # manual, auto
    speed = Column(Integer)
    left_speed = Column(Integer)
    right_speed = Column(Integer)
    connected = Column(Boolean)
    timestamp = Column(DateTime, default=datetime.utcnow, index=True)
```

## 5. 环境异常落库 (`environment_anomalies`)

对应 ORM：`app/models/environment_anomaly.py` · `EnvironmentAnomaly`。

| 字段 | 说明 |
|------|------|
| `id` | 主键 |
| `device_id` | 设备 ID |
| `metric` | 指标：`temperature` / `humidity` / `light` |
| `rule_id` | 规则标识字符串 |
| `peak` | 峰值（可选） |
| `threshold` | 阈值（可选） |
| `detail_zh` | 中文说明 |
| `recorded_at` | 落库时间（UTC naive，查询按窗口筛选） |

索引含 `recorded_at` 及 `(device_id, recorded_at)`。

## 6. 智能助手对话 (`agent_conversations` / `agent_messages`)

对应 ORM：`app/models/agent_chat.py`。

- **AgentConversation**：`id`（UUID）、`title`、`mode`（general/rag）、`created_at`、`updated_at`。
- **AgentMessage**：`conversation_id` 外键、`role`、`content`、`reasoning_content`（可选）、`seq`、`created_at`；级联删除归属会话。
