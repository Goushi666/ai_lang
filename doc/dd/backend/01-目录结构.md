# 后端设计 · 目录结构

```
backend/
├── app/
│   ├── api/
│   │   └── v1/
│   │       ├── sensors.py       # 传感器
│   │       ├── alarms.py        # 告警
│   │       ├── vehicles.py      # 车辆/云台/机械臂/循迹
│   │       ├── devices.py       # 设备
│   │       ├── analysis.py      # 环境分析
│   │       ├── monitoring.py    # 环境监测页 API
│   │       ├── agent.py         # 智能 Agent
│   │       ├── video.py         # 视频代理/配置
│   │       └── admin.py         # 数据维护
│   ├── core/
│   │   ├── config.py
│   │   ├── database.py          # SQLAlchemy 引擎与会话
│   │   ├── security.py
│   │   ├── mqtt.py
│   │   └── timeutil.py          # 时间格式化等
│   ├── models/                  # ORM
│   ├── schemas/                 # Pydantic
│   ├── services/
│   │   ├── sensor_service.py
│   │   ├── alarm_service.py
│   │   ├── vehicle_service.py
│   │   ├── analysis_service.py
│   │   ├── monitoring_service.py    # 监测页温度时间线等
│   │   ├── temperature_forecast.py  # 温度预测（LSTM 等）
│   │   ├── knowledge/               # RAG：嵌入、向量库、分块、检索
│   │   │   ├── embedding_client.py
│   │   │   ├── vector_store.py
│   │   │   ├── chunker.py
│   │   │   └── service.py
│   │   └── agent/               # 智能 Agent（编排）
│   │       ├── service.py
│   │       ├── clarifier.py
│   │       ├── tools/           # BaseTool、各工具实现
│   │       ├── llm/             # client、prompts、reasoning_split
│   │       ├── context/         # SessionManager
│   │       └── skills/          # Skill 注册
│   ├── repositories/
│   ├── websocket/
│   ├── middleware/
│   └── main.py
├── knowledge_docs/            # RAG 用户文档（Markdown）
├── scripts/
│   └── ingest_knowledge.py      # 入库脚本
├── data/vector_db/              # Chroma 持久化（可 .gitignore）
├── requirements.txt
└── .env
```

依赖方向保持 **api → services → repositories**；设计文档以仓库实际文件为准，若有新增路由或服务，同步更新本节。
