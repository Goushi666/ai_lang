# Web 端需求 · 非功能需求

## 1. 性能要求

- 页面首次加载时间 < 3 秒。
- WebSocket 消息延迟 < 200ms（典型网络下目标值）。
- 视频流延迟 < 500ms。
- 支持同时在线用户数 ≥ 10。

## 2. 兼容性要求

- 支持 Chrome、Firefox、Edge 等主流浏览器（最新两个版本）。
- 支持桌面端与平板设备。
- 响应式布局适配不同屏幕尺寸。

## 3. 安全性要求

- 用户身份认证与授权。
- 敏感操作需二次确认。
- 通信数据加密传输（HTTPS/WSS）。
- 防止 XSS、CSRF 等常见 Web 攻击。

## 4. 可用性要求

- 界面简洁直观，操作流程清晰。
- 关键操作提供操作提示。
- 异常情况提供友好的错误提示。
- 支持操作撤销与确认机制。

## 5. 可维护性要求

- 代码结构清晰，模块化设计。
- 关键功能提供注释说明。
- 遵循 Vue.js 最佳实践。
- 提供开发文档与部署文档。
