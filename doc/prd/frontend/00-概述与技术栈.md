# Web 端需求 · 概述与技术栈

## 1. 项目背景

本项目旨在开发井场环境监测与智能巡检系统的 Web 端应用，为用户提供实时环境数据监控、智能巡检车远程控制、视频监控及告警管理等功能。

## 2. 技术栈

- **前端框架**：Vue.js（建议 Vue 3 + Composition API，与实现一致）
- **通信协议**：WebSocket（实时数据）、RESTful API（历史数据查询与配置）
- **后端对接**：FastAPI 服务
