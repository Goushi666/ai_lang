# Web 端需求 · 环境监测看板

## 1. 实时数据展示

- 显示温度、湿度、光照强度等环境参数的实时数值。
- 数据通过 WebSocket 实时推送，无需手动刷新。
- 支持数据单位显示（℃、%RH、Lux 等）。
- 尚无后端数据时，界面应展示明确空态（如 `--`），不展示虚构默认曲线或占位真值（与后端「无数据」策略一致）。

## 2. 历史数据查询

- 支持按时间范围查询历史环境数据。
- 提供数据趋势图表展示（折线图）。
- 支持数据导出功能（CSV 格式）。

## 3. 数据可视化

- 使用图表组件展示数据变化趋势。
- 支持多参数对比显示。
- 提供时间轴缩放与拖拽功能（与所选图表库能力对齐）。

## 4. 与后端依赖

- REST：`GET /api/sensors/latest`、`/history`、`/export`。
- WebSocket：订阅 `sensor_data` 类型消息。
