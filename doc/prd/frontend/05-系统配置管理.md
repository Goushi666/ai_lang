# Web 端需求 · 系统配置管理

## 1. 设备管理

- 查看已接入设备列表。
- 显示设备在线状态。
- 支持设备基本信息配置。

## 2. 告警阈值配置

- 配置各环境参数的告警阈值。
- 支持多级阈值设置。
- 配置修改后实时生效（以后端实际生效策略为准）。

## 3. 用户设置

- 用户登录/登出功能。
- 个性化界面设置（主题、语言等）。
- 通知偏好设置。

## 4. 与后端依赖

- 设备：`GET /api/devices`、`GET /api/devices/{id}`、`PUT /api/devices/{id}`（以实际开放接口为准）。
- 告警阈值：`GET/PUT /api/alarms/config`。
