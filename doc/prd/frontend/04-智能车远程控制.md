# Web 端需求 · 智能车远程控制

## 1. 手动控制模式

- 提供方向控制按钮：前进、后退、左转、右转、停止。
- 支持键盘快捷键控制（WASD 或方向键）。
- 显示当前控制模式状态（手动/自动）。

## 2. 自动循迹模式

- 提供自动循迹模式启动/停止按钮。
- 显示循迹状态指示（运行中/已停止）。
- 显示循迹偏差量与修正量（可选）。

## 3. 运行状态监控

- 实时显示车辆速度信息。
- 显示当前运行模式（手动/自动）。
- 显示指令执行反馈状态。
- 显示连接状态（在线/离线）。

## 4. 安全保护

- 断连状态下禁用控制按钮。
- 显示异常状态告警信息。
- 提供紧急停止按钮（优先级最高）。

## 5. 与后端依赖

- REST：`POST /api/vehicle/control`、`GET /api/vehicle/status`（路径前缀与 OpenAPI/部署一致）。
- WebSocket：订阅 `vehicle_status` 类型消息。
