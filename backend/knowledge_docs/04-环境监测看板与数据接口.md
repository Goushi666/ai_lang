# 环境监测看板（客户端操作）

## 这一页能干什么

- 看 **当前** 温度、湿度、光照等实时数（单位如 ℃、%RH、Lux）。
- 选 **时间段** 看这段时间内的 **历史曲线**（可多参数一起看，具体以页面为准）。
- 有的版本支持图表 **缩放、拖拽** 时间轴，方便放大某一段细看。
- **导出**：把所选时间范围内的数据导出成表格文件（常见为 CSV），在浏览器里下载保存。

## 怎么操作（通用顺序）

1. 打开左侧 **「环境监测」**（名称以实际菜单为准）。
2. 看顶部或卡片区的 **实时数值**；无数据时一般会显示 `--` 或空态提示，属正常现象。
3. 需要历史时：先选 **开始时间、结束时间**（或快捷「最近 24 小时」等），再点 **查询 / 刷新**。
4. 需要导出时：时间选好后点 **导出**，按浏览器提示保存文件。

## 和环境分析的关系

同一套环境数据上，有的版本在监测页还提供 **「环境分析」** 入口或页签，用统计、摘要、异常列表等方式补充曲线；详见 **`08-环境分析模块.md`**。
