# 告警中心（客户端操作）

## 这一页能干什么

- 看 **最新告警** 和 **历史告警**。
- 按 **级别、时间范围** 等条件筛选（以页面筛选项为准）。
- **导出** 告警列表到文件，便于留档或发邮件。
- 若有 **删除** 权限，可清理已处理记录（删除前请确认公司制度）。

## 怎么操作

1. 打开左侧 **「告警中心」**。
2. 列表默认多为 **时间倒序**（新的在上）；点某一条可看详情（若支持）。
3. 需要筛选时：先选 **级别、起止时间** 等，再点 **查询 / 应用**。
4. 需要导出时：筛选范围选好以后点 **导出**，在浏览器里保存下载的文件。

## 通知方式（若页面提供）

- **弹窗**：新告警出现时的页面提示。
- **浏览器通知**：首次使用浏览器可能问你是否允许通知，允许后可在后台标签页也收到提醒。
- **声音**：可在设置里开关提示音，避免夜间打扰。

## 告警阈值在哪里改

温度、湿度、光照等 **超过多少算告警**，一般在 **「系统设置」** 里配置；详见 **`09-系统设置与设备管理.md`**。告警中心主要负责 **查看与处理记录**。

## 时间显示

列表里的时间一般以 **北京时间** 展示，便于和现场值班表对照（具体以你单位部署为准）。
