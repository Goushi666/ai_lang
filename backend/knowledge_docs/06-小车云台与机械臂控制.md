# 小车、云台与机械臂（客户端操作）

均在 **「巡检与遥控」**（或同类名称）页面中操作，具体布局以实际界面为准。

## 手动控制小车

- 用 **前进 / 后退 / 左转 / 右转 / 停止** 按钮控制；还支持键盘 **WASD** 或 **方向键**。
- 页面上会显示 **是否在线、当前速度、当前模式**；**离线或未连接时按钮会变灰**，此时不要强行连点。
- **急停**：红色或醒目的急停按钮，一按即停，优先级最高。

## 自动循迹

- 有 **循迹开/关**（或「自动循迹」）类开关；打开后小车按线路自动跑，页面上会显示循迹 **运行中 / 已停止**。
- **重要**：循迹 **打开时**，通常 **不能再手动** 调方向、云台等（避免和自动逻辑冲突）。要先 **关闭循迹** 或切回 **手动模式**，再调云台、机械臂或方向。

## 云台（抬头/左右看）

- 用滑块、旋钮或角度输入控制 **水平、俯仰** 两轴（界面上可能标为两路角度）。
- 按页面标注的 **安全范围** 调节，不要一次拧到底；若提示「循迹中不可用」，请先按上一节退出循迹。

## 机械臂

- 用各 **关节** 的滑块或数值逐个调节（常见为多个关节一条链）。
- 同样建议 **小步调整**，观察画面里机械臂运动再微调；遇「不可用」提示时先看是否在循迹或急停状态。

## 安全与现场规范

- 遥控前确认现场 **无人站在车与机械臂活动范围内**。
- 出现异常声音、冒烟、指令无响应等情况，立即 **急停** 并通知现场负责人。
