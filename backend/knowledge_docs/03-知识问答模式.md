# 知识问答模式（客户端说明）

## 做什么用

在智能助手页切换到 **「知识问答」** 后，助手会优先根据系统内置的操作说明来回答，适合：

- 各菜单、按钮是干什么的  
- 巡检、小车、云台、机械臂、告警、环境分析等 **怎么用**  
- 使用上的注意点（例如循迹时不能手动遥控等）

查实时温度、查历史曲线、导出数据等，更建议用 **「通用对话」** 或直接到 **环境监测 / 告警中心** 页面操作。

## 你需要做什么

1. 进入 **智能助手**。  
2. 点右上角 **「知识问答」**。  
3. 用日常中文描述问题即可；若回答里引用了某篇说明，会带文档名，便于你对照。

说明内容随系统版本更新；若你发现说明和实际界面不一致，请向管理员或开发反馈以便更新文案。
