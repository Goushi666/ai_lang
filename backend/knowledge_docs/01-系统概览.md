# 井场环境监测与智能巡检系统 · 概览

## 系统是做什么的

面向井场运维：查看环境数据、处理告警、用智能助手问问题、看视频画面并遥控小车/云台/机械臂。

## 左侧菜单怎么用

从左到右进入各模块（当前选中的菜单会高亮）：

1. **环境监测**：实时温湿度光照、历史曲线、预测、异常片段、数据导出。→ **`04-环境监测看板与数据接口.md`**
2. **智能助手**：打字提问、管理对话、选通用对话或知识问答。→ **`02-智能助手与环境数据.md`**
3. **巡检与遥控**：车载视频、车辆状态、方向控制、循迹、云台与机械臂。→ **`05-视频巡检与车画面.md`**、**`06-小车云台与机械臂控制.md`**
4. **告警中心**：告警列表、筛选、导出、删除等。→ **`07-告警中心与阈值配置.md`**
5. **系统设置**：设备、告警阈值、数据维护等。→ **`09-系统设置与设备管理.md`**

**环境分析**（汇总统计与异常摘要，和看板里的曲线互补）：→ **`08-环境分析模块.md`**

知识问答使用方式见 **`03-知识问答模式.md`**。

## 使用提示

- **循迹**打开时，部分手动遥控会被锁住，要先退出循迹再调云台或方向，详见 **`06-小车云台与机械臂.md`**。
