# 井场环境监测与智能巡检系统 · 概览

## 系统名称与用途

本系统面向井场场景，提供环境监测、智能助手、巡检与遥控、告警中心及系统设置等能力，帮助运维人员查看实时数据、分析异常、远程查看车载视频并控制小车/云台/机械臂。

## 五大功能模块

1. **环境监测**：实时温湿度光照、曲线与预测、24 小时异常片段列表、数据导出。
2. **智能助手**：自然语言查询数据与告警、环境分析摘要；支持会话管理与「深度思考」展示（若模型提供）。
3. **巡检与遥控**：视频流（HLS/MJPEG）、运行状态、小车方向控制、云台与机械臂关节控制、循迹模式切换。
4. **告警中心**：异常记录查询、筛选、导出与删除（界面时间一般为北京时间展示）。
5. **系统设置**：告警阈值、数据库维护（清空采样/异常记录）等。
