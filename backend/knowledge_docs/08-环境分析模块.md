# 环境分析（客户端操作）

## 和环境监测看板有什么不同

- **环境监测**：偏实时数、历史 **曲线**、导出原始点。
- **环境分析**：在同一套数据上做 **汇总**（如最高、最低、平均）、**文字摘要**、**异常片段列表**，帮你快速读结论。

两者互补，不是重复；入口可能在监测页的 **页签/按钮**，或左侧独立菜单，以实际界面为准。

## 这一页能干什么

1. 先选好 **时间范围**（可与看板默认一致，如最近 24 小时）。
2. 点 **分析 / 重新分析 / 刷新** 等按钮，等待加载结束。
3. 查看页面上的：
   - **数字汇总**（各指标统计）；  
   - **可选图表**（如按小时分布）；  
   - **文字结论**（如「温度整体平稳」等）；  
   - **异常列表**（若有）：类型、时间段、设备等。
4. 若有 **「导出异常」**：在选好时间范围后导出表格，保存到本机。

## 没有数据时

会显示 **空态或提示**，不会编造假数字；可换一个时间段或确认现场设备是否在采样。

## 和智能助手联动（若页面有按钮）

有的版本提供 **「用助手解读」**：会把当前分析结果或时间范围带到智能助手，你继续用自然语言追问即可；进入后操作方式见 **`02-智能助手与环境数据.md`**。
