# 系统设置（客户端操作）

## 从哪里进入

左侧主导航 **「系统设置」**（名称以实际菜单为准）。

## 常见能配什么

### 告警阈值

- 设置温度、湿度、光照等 **超过多少触发告警**（可能分多级，以表单为准）。
- 改完后一般点 **保存**；保存成功页面会有提示。具体何时生效以你单位系统说明为准。


### 数据维护（若有权限才可见）

- 例如 **清空历史采样、清空异常记录** 等，通常属于 **危险操作**，会要求 **二次确认**。
- **执行前务必确认** 公司是否允许、是否已备份；误删可能无法恢复。


## 和告警中心的关系

阈值在 **系统设置** 里改；**告警中心** 里看触发的记录。改阈值不会自动删掉旧告警记录。
