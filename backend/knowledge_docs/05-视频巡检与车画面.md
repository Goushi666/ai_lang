# 视频巡检与车载画面（客户端操作）

## 这一页能干什么

- 看巡检车 **实时摄像头画面**。
- **播放 / 暂停**、**全屏**（以页面按钮为准）。
- 看 **连接状态**（是否在线、是否在缓冲等）。

## 怎么操作

1. 打开左侧 **「巡检与遥控」**（或带「视频」「巡检」字样的菜单，以实际为准）。
2. 在视频区域等待画面出现；若长时间黑屏或转圈，先看状态提示，再检查网络或联系管理员。
3. 需要全屏时点 **全屏** 图标；退出全屏按 **Esc** 或浏览器全屏退出方式。
4. 若画面卡顿，可尝试缩小浏览器窗口、关闭其他占网速的应用，或换网络环境。

## 使用注意

- 画面延迟与 **现场网络、车机性能** 有关，界面上的「流畅」程度会随环境变化。
- 若公司规定禁止录屏或外传画面，请遵守现场安全与保密制度。
